# Tradyfi (Zerodha Clone)

Welcome to Tradyfi, a modern investment and trading platform clone. 

## Mobile Responsiveness Updates

We have recently improved the mobile responsiveness of the entire platform:

- **Mobile Navigation Menu (Frontend)**: A sleek hamburger drawer overlay for screens < 768px.
- **Mobile Bottom Navigation (Dashboard)**: Native-feeling bottom tab bar for easy screen switching (Dashboard, Watchlist, Orders, Holdings, Funds).
- **Responsive Data Grids & Tables**: All tables scroll horizontally on small viewports with smooth touch support.
- **Stacked Layouts**: Optimized grid layouts for cards, stats, and widgets to prevent horizontal overflow.
- **Responsive Order Ticket Window**: Repositioned buy/sell modal aligned above the bottom nav for mobile viewports.
